# Cricket_data_analytics
# Super 11 for T20

The aim of the project is to select 11 players based on their performance in 2022 T20 world cup 


## Introduction

The Fantasy Cricket Team Selector with Power BI is a dynamic data analytics project that empowers cricket enthusiasts to craft their fantasy cricket team by leveraging the power of visualizations and insights. Focused on the 2022 T20 World Cup, this Power BI solution utilizes historical data to guide users in making strategic decisions for their fantasy teams.

## About files


- 1.Attached the data collected from https://www.espncricinfo.com/ in "t20_json_files"
- 2.Also,added the converted csv format from json format in folder "t20_csv_files"
- 3.Provided the python code for data preprocessing and csv conversion in "Data_preprocessing.ipynb".
- 4.Provided the criteria for selecting the player in "Selection Criteria"
- 5.Provided the Various DAX measures used for Selection in power bi in "DAx measures and calculated fields"
- 6.Provided the Final dashboard of Player selection in "project.pbix"

## Getting Started

1.Download all files and unzip it.
2.Use the "Data_preprocessing.ipynb" for data preprocessing and csv conversion.
3.Create the measures using "DAx measures and calculated fields" in power bi.
4.Use "project.pbix" as reference to create dashboard.

### Prerequisites

1.Jupyter Notebook.
